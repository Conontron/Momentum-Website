What makes it special?

Features

We include most apps in the firmware, as long as they work and serve an interesting purpose. We also add the best features from other custom firmware, and make our own exclusives.

Stability

We spent many hours rewriting core parts of the firmware as well as some of its apps and addons to ensure stability. A task that was long needed, so we tackled it right away.

Customization

Want to change the animations, turn on/off some icons, change your flipper's name? You absolutely can. No need to mess with code, its all done right on your device with a convenient app.

Main Features

Asset Packs

Easily install and switch between animation and icon sets, with no coding required. Pick between dozens of community packs, or make your own.

Bad Keyboard

Expands on flipper's BadUSB functionality by allowing connection over bluetooth, with full control over name and MAC address.

Countless Protocols

Many added protocols for SubGhz signals, including support for rolling code devices. Multiple additional types of NFC cards and tags are also supported.

New Interface

Completely redesigned interface from the ground up: main menu as a 'wii-style' app grid, handy control center with quick toggles, and more.

Xtreme Settings

The one app to rule them app, it lets you customize all our exclusive features easily and in great detail. Also handles procol settings on-device.

Level System

An optimized backend for levelling that features levels from 1 to 30, with each one unlocking a new animation. Also supports assets packs.
